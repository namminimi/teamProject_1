부분별로 함수 분해 했을때 조건 나누기/ 조건 더 늘려도 헷갈리지 않을거같다
타자가 스윙 성공했을때

1루 함수 {
현 좌표에 아무도 없을 경우 스윙 성공 1 베이스진출
/ 볼4개받을 경우 1 베이스 진출
/현 좌표에 있고 다음 좌표에 아무도 없을 경우  스윙 성공 현 좌표 타자는 2베이스 진출 과 동시에 1 베이스 진출 
/ 볼 4개 받을 경우/
}

2루 함수 {
다음 좌표에 아무도 없을 경우 다음 타자 스윙 성공할 경우 1 베이스진출
/다음 좌표에 있고 다음 타자 스윙 성공할 경우 1 베이스진출/ 3루에 있는 타자도 1베이스 진출(점수)
/다음 좌표. 전 좌표에 아무도 없을 경우 볼 4받아도 가만히
/다음 좌표는 없고 전 좌표에 타자 있을 경우 볼 4받은 후 1베이스 이동
}

3루 함수 {
다음 타자 스윙 성공할 경우 1 베이스진출(점수)
/전 좌표에 아무도 없을 경우 볼 4받아도 가만히
/전 좌표에 타자 있을 경우 볼 4받은 후 1베이스 이동(점수)
}

포수쪽 함수{
스윙성공 했을 때 1루 베이스 이동
스윙성공 했을 때 1루 베이스 이동, 1루좌표에 타자있을경우 1루타자는 2루로 이동
볼 4개 받을 경우 1루 베이스 이동
볼 4개 받을 경우 1루 베이스 이동, 1루좌표에 타자있을경우 1루타자는 2루로 이동
}
